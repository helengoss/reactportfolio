# Helen Goss – React Portfolio

A personal portfolio built with React as part of a full-stack bootcamp (Project 9, Week 9). It showcases the projects completed across the course and includes an interactive carousel gallery and a working contact form.

---

## Features

- Multi-page layout with sidebar navigation (Home, Portfolio, Contact)
- Interactive carousel gallery displaying 9 bootcamp projects, each with a description, live site link, and GitHub link
- Working contact form powered by Formspree
- Fully responsive design
- Built with plain CSS — no frameworks

---

## Technologies Used

- [React](https://react.dev/) (v18)
- [Vite](https://vitejs.dev/) (build tool)
- Plain CSS
- [Formspree](https://formspree.io/) (contact form handling)

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/helengoss/project9.git
   cd project9
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run locally:
   ```bash
   npm run dev
   ```

4. Open your browser at `http://localhost:5173`
